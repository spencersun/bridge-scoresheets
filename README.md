Bridge scoresheets, inspired by a similar scoresheet from C&T Bridge Supplies.

Coded by hand in raw PostScript (I have a weird idea of fun) in 1994 and posted
to rec.games.bridge; updated minimally in 2014 and added to github.

Save trees, re-use old paper and print these on the backs!
